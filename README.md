# Discounts Side Cart

A React + TypeScript implementation of a discounts side cart system based on the provided Figma design.

## Features

- Add manual discounts
- Edit discounts
- Delete discounts
- Enable / disable discounts
- One-time discounts
- Monthly discounts
- Percentage and fixed (€) discounts
- Duration support for monthly discounts
- Dynamic overview calculations
- Responsive and reusable components
- Figma-inspired UI

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
